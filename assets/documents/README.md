# KACP 資料フォルダ

このフォルダには、各プランの資料PDFを配置します。

## 必要なファイル

- `kacp_business.pdf` - KACP（事業者向け）の詳細資料
- `kacp_home.pdf` - KACP HOME（家庭向け）の詳細資料

## 配置方法

1. 用意したPDFファイルをこのフォルダに配置
2. ファイル名は上記の通りに設定
3. ページのダウンロードボタンから自動的にリンクされます

## 注意事項

- PDFファイルが存在しない場合、ダウンロードは404エラーになります
- ファイルサイズは適切に圧縮してください（推奨: 5MB以下）
